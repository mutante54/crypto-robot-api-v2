## Description

API v2 desenvolvida sobre o NodeJS (NestJS/ Javascript) para atender ao consumo de dados do front-end da aplicação Crypto Robot.

Projeto baseado em:

[Nest](https://github.com/nestjs/nest) framework JavaScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Troubleshooting

Em caso de erro ao iniciar a aplicação:

> *npm ERR! nest-babel-starter@1.0.0 start: `babel-node index.js*

# Motivo do erro: 

Possível incompatibilidade de versões entre uma ou mais libs do projeto com relação ao ambiente de desenvolvimento local (Node/ NPM). Logo, o problema pode ocorrer com qualquer dependência do projeto, principalmente se a mesma já tenha sido instalada no cache global anteriormente.

> Atenção: os passos abaixo devem ser executados apenas em último caso. Eventuais updates em versões de libs do projeto podem gerar bugs na aplicação!.

1. Delete o arquivo `package-lock.json` da raiz do projeto;
2. Delete o diretório `node_modules` da raiz do projeto;
3. Execute o comando `npm cache clean --force`;
4. Execute o comando `npm install`na raiz do projeto.
